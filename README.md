# description
숨넘어가는 컬리 검색 결과를 조금더 보기 편하게 수정한 스크립트.

네이버 쇼핑에서는 그럭저럭 단위당 가격 ( 100원/gram 같은 )을 표시해주는 반면, 
컬리는 의도적으로 이런 구분자를 없엔거같음. 
쿠팡에선 이런걸로 승질 안났는데, 탈팡하고보니 당일배송 선택지가 많이 없어서 컬리를 선택하게 되었는데 여간 빡칠수가 없음.
그래서 급하게 스크립트를 만들어, 단위당 가격을 낼 수 있는 상품들을 확인할수 있게 처리하는 스크립트를 만듬.


# how to use
tampermonkey 같은 커스텀 스크립트 인젝터에 해당 js파일을 인식시키면 끝.

이제 컬리에서 검색을 사용할 경우 품목 하단에 가격이 나옴.
단, 상품내 옵션질에 따라 여러 상품이 존재할수 있기때문에, 여러 상품이 있는경우 각각 상품의 그람당 가격을 표기함.

그런데 여기서 각각 상품의 이름등 정보를 추가하자니 화면간섭이 너무 커서, 그냥 가격만 표기하게 둠. 어떤 옵션이 해당 가격인가는 애석하게도 직접 들어가서 봐야함.
